Projects

Software Público Brasileiro (Brazilian Public Softwares) - Software Público Brasileiro (SPB) is a colaborative platform for open source software development to the brazilian public administration. SPB stores all open source projects used through all public organization. It's integrates with Gitlab, Colab and Noofero applications.
Sistema de Gestão de Desempenho (Performace Management System) - Sistema de Gestão de Desempenho (SGD) is a system that provides tools to manage a team performace based on their scores given by others team meambers. It's build with Ruby on Rails Fullstack. It follows the local requirements and rules of the UnB (Brasília University).
